That Offset Image Grid From
The Royal Tenenbaums
. Javascript

<figure
  data-offset = "wes1.png"
></figure>
      A brand-new javascript file for repeating an HTML element's background image in a manner such that there will always be a vertically and horizontally centered image, and adjacent columns will vertically offset by exactly one-half of the image's height.

Available on Github, built by Tyler.

3 Ways To Use

1HTML + Inline URL

<figure
  data-offset = 'wes1.png'
></figure>

2HTML + CSS URL

figure{
  background:url('wes1.png')
}

<figure
  data-offset
></figure>

3HTML + Parameters

##Default parameters
"src"       : "",

String of the image's URL


"snap"      : false,

Change width of image to create a whole number of columns, only respecting one min/max property


"minWidth"  : 0,
"maxWidth"  : 0,
"width"     : 0,
"minHeight" : 0,
"maxHeight" : 0,
"height"    : 0

Can be a number or string ending in 'px' or '%'

##Override in HTML
<figure
  data-offset = '{
    "src"      : "wes1.png",
    "width"    : "20%",
    "maxWidth" : 200,
  }'
></figure>

2 Steps To Run

1Link The Javascript File

<script
  src="ThatOffsetImageGridFromTheRoyalTenenbaums.js"
></script>
