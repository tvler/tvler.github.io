That Offset Image Grid From
The Royal Tenenbaums
. Javascript

<figure
  data-offset = "wes1.png"
></figure>
      A brand-new javascript file for repeating an HTML element's background image in a manner such that there will always be a vertically and horizontally centered image, and adjacent columns will be vertically offset by exactly one-half of the image's height.

Available on Github, built by Tyler.

3 Ways To Use

1HTML + Inline URL

<figure
  data-offset = 'wes1.png'
></figure>

2HTML + CSS URL

figure{
  background:url('wes1.png')
}

<figure
  data-offset
></figure>

3HTML + JSON

<figure
  data-offset = '{
    "src"      : "wes1.png",
    "width"    : "20%",
    "maxWidth" : 200
  }'
></figure>
##Default values
"src"       : "",

String of the image's URL
(If not given, CSS URL will be used)


"snap"      : false,

Change width of image to create a whole number of columns, only respecting one min/max property


"minWidth"  : 0,
"maxWidth"  : "100%",
"width"     : 0,
"minHeight" : 0,
"maxHeight" : "100%",
"height"    : 0

Can be a number or string ending in 'px' or '%'

2 Steps To Run

1Link The Javascript File

<script
  src="ThatOffsetImageGridFromTheRoyalTenenbaums.js"
></script>
##This creates the variable_offsetImg

2Set An Event To Update The Grid

_offsetImg.update()

When a data-offset element resizes, it's grid needs to update

##Example hook
window.onresize =
  _offsetImg.update;

Settings

1Add, Edit, Or Remove Data-Offset Elements

_offsetImg.init()

Run this function after an element is added, edited, or removed

2Change Dafault Values

_offsetImg.defaultValues

The exposed object's properties can be modified, which may reduce overall markup for some situations

If it's modified after DOMContentLoaded, run _offsetImg.init()
