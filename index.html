ThatHalfDropImageGridFromTheRoyalTenenbaums.js

2015

My first js library, initally created for a Cal Poly Library project.

Made to repeat an HTML element's background image in a centered half-drop pattern, like that thing (1, 2, 3) from that movie. Each pattern is extensively customizable through JSON data attributes, with properties based off CSS naming standards.

It seemed funny to make the name so long, because I couldn't think of an easier way to describe it to people for the entire time I worked on it. Please send a pull request if you think of a more concise description.

View the project website.

Mapping crowdedness

2015

My senior project, a web app that visualized crowdedness inside buildings using the placement and activity of wifi routers as data points.

My project was an extension of a map application of my school’s library, existing as a visibly toggled “layer” of the map similar to traffic and satellite layers in Google Maps.

For every floor of the building, a Voronoi Diagram is generated by the coordinates of the floor’s wifi routers, and each polygon is assigned a “crowdedness” level based off how many people are connected to it. The polygons are abstracted into a user-facing layer of fog over the map.

The denser the fog, the more crowded the Voronoi polygon is beneath it.

Voronoi diagram off the library's wifi routers
Visualizing crowdedness by fog density

My final work is the published paper "Visualizing Population Density Based On Wifi Router Location and Network Usage."

Inspired by the talk "User Interface Algorithms" by Mark DiMarco at JSConf2014.

Community exhibits

2015

I redesigned the web presence of Cal Poly Library's exhibition spaces, art galleries of student and faculty creative gallery work.

The goal of the Community Exhibits redesign was to make the site as engaging as actual exhibit content, bringing more attention to the gallery work, and encouraging more community members to use our facilities.

I created two main items: an interactive exhibit calendar, and exhibit cards with information on a particular space. Both items were adapted by the library to be default elements for future projects.

I discovered Cal Poly Library’s amazing design team through participating in a community exhibit myself, so this project was special to me.

Pizza Club

2015

Pizza Club was prototyped as a mobile web app at Cal Poly's 2015 Hackathon and won Best Design. I had the initial concept and did design and frontend development.

Pizza Club is a secret social network at Cal Poly that lets users know when free pizza is available on campus. The info is obtained by signing up for every club with a fake email address and filtering the received messages for mentions of "pizza" at an event.
